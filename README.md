# NotebookLM - Agente Especialista: Auditor de Dados em Saúde

Este repositório contém a base de conhecimento estruturada para o desenvolvimento de um Agente Especialista em Auditoria de Dados em Saúde, utilizando o Google NotebookLM.

## Objetivo
Consolidar fontes oficiais para subsidiar auditorias técnicas, cruzando normas regulatórias com dados públicos de saúde.

## Estrutura do Projeto
- `00-index/`: Índice central com a lista de fontes (sources_master.csv).
- `01-regulacao/`: Normas, RDCs, resoluções e guias técnicos.
- `02-dados-vigilancia/`: Bases abertas, dicionários e manuais de sistemas (DATASUS).
- `03-auditoria-governanca/`: Relatórios de fiscalização (TCU) e manuais de auditoria.
- `04-metodos-heor/`: Métodos de avaliação econômica e diretrizes (ISPOR/CONITEC).

## Como utilizar
1. Utilize o `sources_master.csv` para gerenciar as fontes.
2. Cada pasta temática contém o material específico para os notebooks do NotebookLM.
